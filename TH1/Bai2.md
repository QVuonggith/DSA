Mô phỏng từng bước

Cho mảng A = [7, 3, 9, 12, 5, 8, 1] và x = 5.

Trường hợp chỉ số i bắt đầu từ 0:

Bước 1: Chỉ số i = 0, Giá trị A[0] = 7. So sánh: 7 khác 5. Kết luận: Sai, tiếp tục tăng i.

Bước 2: Chỉ số i = 1, Giá trị A[1] = 3. So sánh: 3 khác 5. Kết luận: Sai, tiếp tục tăng i.

Bước 3: Chỉ số i = 2, Giá trị A[2] = 9. So sánh: 9 khác 5. Kết luận: Sai, tiếp tục tăng i.

Bước 4: Chỉ số i = 3, Giá trị A[3] = 12. So sánh: 12 khác 5. Kết luận: Sai, tiếp tục tăng i.

Bước 5: Chỉ số i = 4, Giá trị A[4] = 5. So sánh: 5 bằng 5. Kết luận: Đúng, tìm thấy x! Dừng thuật toán.
=> Giá trị hàm trả về là: 4
